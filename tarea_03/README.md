## **Tarea 03: Pieza narrativa individual**

**Link página web:** https://emiliavalenzuela.github.io/tareas-emiliav/

**Proceso**

Para esta tarea usé la crónica y la visualización que desarrolle en previamente en el proyecto sobre natalidad en Chile. El obetivo fue construir una página web clara usando las herramientas HTML.

Pirmero cree la carpeta *docs* en mi repositorio de tareas, para luego agregar *index.html* y así poder empezar a crear mi página web. 

La estructura del sitio se organizó mediante herramientas como *head, header, div, h1, h2, p, img, a, body y footer*. Además, agregué una lista con los principales hallazgos para complementar la visualización. 

El foco pricnipal estuvo en ordenar el contenido de manera cohoerente, para que se entendiera fácil y así facilitar la lectura.
